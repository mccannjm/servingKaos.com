The Kernel

Three primitives. That's the whole engine.

  • A weighted graph — nodes gain weight when revisited, decay when abandoned. What you return to gets heavier. What you release fades.
  • A ring classifier — every character classified into a ring in microseconds. The colors appear as you type.
  • A version heartbeat — every mutation pulses. The constellation watches the pulse and breathes.

What It Looks Like

You type bracketed text. The rings appear in color — a mirror of what you wrote, classified. A constellation shows your thinking as a node graph. Nodes orbit a nucleus. Weight pulls them inward. Pins glow red and never decay.

The bracket notation is a protocol, not a feature. It works in any text editor, any conversation, any surface. Zeus just makes it visible.

Try It

No install needed. Runs in your browser.

The Native Instrument

Zeus is a macOS app built with Swift and Metal. The web version above runs the same kernel — same classifier, same layout math, same colors. The native instrument adds persistence (Heimdall), constellation intelligence (Claude API), temper (session mood), and the full door system.
